# How to run?
- Clone repository
- Have Docker running on the background
- Start backend. This should also start Postgres DB and seed some data into it.
    - Backend runs at http://localhost:3000
- Start frontend
    - Run `npm ci`
    - Run `npm run dev`
    - Frontend runs at http://localhost:8080
